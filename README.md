This console application is meant to gather, calculate and classify membership plans.
The classes contained within are:
- Main;
- MembershipPlan (Abstract);
- GymMembershipPlan;
- PersonalTrainingPlan;
- OnlineCoachingPlan;
- CorporateWellnessPlan;
- RemoteAccess (interface);
- Billable (interface);
- Freezable (interface).
